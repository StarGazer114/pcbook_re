# 赛博扫盲手册 (Cyber Literacy Manual) - 项目指南

本项目是一个基于 [VitePress](https://vitepress.dev/) 构建的数字素养入门手册。其核心目标是使用通俗易懂的语言、生活化的类比，帮助对电脑及数字技术感到困惑的初学者（特别是更习惯使用手机的读者）建立起系统的数字世界认知。

## 🚀 项目概览

- **定位**：面向零基础读者的科普手册。
- **技术栈**：VitePress, Vue.js, Markdown。
- **核心特色**：
  - **零基础友好**：大白话解释复杂概念。
  - **安全意识**：强调网络安全与防骗。
  - **紧跟前沿**：涵盖 AI、云计算、区块链等话题。

## 🛠️ 构建与运行

在项目根目录下，你可以执行以下命令：

- **安装依赖**：
  ```bash
  npm install
  ```
- **启动开发服务器**：
  ```bash
  npm run docs:dev
  ```
  默认在 `http://localhost:5173` 运行，支持热更新。
- **构建静态站点**：
  ```bash
  npm run docs:build
  ```
  生成的静态文件将存放在 `.vitepress/dist` 目录下。
- **预览构建结果**：
  ```bash
  npm run docs:preview
  ```

## 📂 目录结构规范

- **`docs/`**：存放手册的核心内容，采用 Markdown 编写。
  - `index.md`：项目首页配置。
  - `preface.md`：前言。
  - `ch[N]-[name].md`：各章节内容（如 `ch1-what-is-computer.md`）。
  - `appendix.md` / `afterword.md`：附录及后记。
- **`.vitepress/`**：存放 VitePress 的配置文件、主题及静态资源。
  - `config.mts`：站点配置文件，定义侧边栏、导航栏等。
- **`node_modules/`**：项目依赖包。

## ✍️ 开发建议与约定

1. **写作风格**：
   - **口语化**：尽量避免使用过于生涩的专业术语，必须使用时请进行生活化类比。
   - **图文并茂**：建议每章都配备相关的示意图或截图辅助理解。
   - **互动性**：每章末尾应附带 3~5 道简单的测验题，帮助读者巩固知识。
2. **文档修改**：
   - 如果新增或修改了文档文件名，请同步更新 `.vitepress/config.mts` 中的 `sidebar` 配置。
   - 保持 Markdown 标题层级的严谨性。
3. **贡献流程**：
   - 建议按照章节顺序进行内容增补。
   - 修改内容后，请务必运行 `npm run docs:dev` 进行本地预览，确保排版和链接正确。
