# 赛博扫盲手册 (Cyber Literacy Manual)

🚀 **从零开始，读懂数字世界** —— 写给每一个想要拥抱科技、却不知从何开始的你。

本手册旨在帮助那些平时更多使用手机、但在面对电脑和数字技术时感到困惑的朋友们。通过生活化的类比和直观的操作指南，我们带你一步步建立起坚实的数字技能基础。

## 🌟 核心特色

- **🖥️ 零基础友好**：全书不预设任何计算机知识，用大白话解释复杂概念。
- **🛡️ 安全意识先行**：从密码管理到识别诈骗，帮你建立坚固的数字安全防线。
- **🤖 紧跟时代前沿**：涵盖人工智能、云计算等热门话题，让你不再被新概念“唬住”。
- **🎯 学以致用**：配有实操指南，手把手教你搞定数字生活的方方面面。

## 📖 内容概览

本手册共包含十个核心章节及两份附录，全面覆盖了从硬件、软件到网络及前沿技术的扫盲知识：

- **引导篇**
  - 前言：为什么要读这本书
- **第一部分：基础认知**
  - 第一章：计算机是什么？——认识这台“聪明的盒子”
  - 第二章：硬件世界——计算机的“五脏六腑”
  - 第三章：软件世界——让机器“听话”的语言
- **第二部分：网络与数据**
  - 第四章：网络世界——连接一切的“蜘蛛网”
  - 第五章：数据与信息——数字世界的“原材料”
  - 第六章：网络安全——数字世界的“防盗门”
- **第三部分：智能与生活**
  - 第七章：人工智能（AI）——会“思考”的机器
  - 第八章：数字生活实用指南——学以致用
- **第四部分：进阶与未来**
  - 第九章：编程与代码——和机器说话的语言
  - 第十章：未来已来——前沿技术一览
- **补充篇**
  - 附录：术语表、学习资源、快捷键与安全自查清单
  - 后记：保持好奇心，持续学习

## 🛠️ 本地开发与预览

本项目基于 [VitePress](https://vitepress.dev/) 构建。

### 1. 安装依赖
在项目根目录下运行：
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run docs:dev
```
启动后，访问浏览器中的 `http://localhost:5173` 即可预览排版好的图文文档。

### 3. 构建静态站点
```bash
npm run docs:build
```

### 4. 预览构建后的版本
```bash
npm run docs:preview
```
