## 📐 编写建议

| 建议项 | 说明 |
|---|---|
| **语言风格** | 口语化、多用类比，避免堆砌术语 |
| **配图原则** | 每个概念配一张示意图，图文比约 1:3 |
| **章节长度** | 每章控制在 3000～5000 字，适合碎片化阅读 |
| **知识检验** | 每章末尾附 3～5 道小测验，加深记忆 |
| **实操练习** | 涉及操作类内容，提供分步截图教程 |
| **版本维护** | 建议每年更新一次，跟进技术变化 |
