# 第六章：网络安全——数字世界的"防盗门"

数字世界虽然精彩，但也潜伏着各种“赛博小偷”和“骗子”。在这个没有警察随时巡逻的虚拟世界里，你必须学会为自己装上防盗门。

### 6.1 常见网络威胁：它们是怎么害人的？

*   **病毒（Virus）**：像生物病毒一样，会感染其他文件，破坏你的电脑系统，导致死机或文件损坏。
*   **木马（Trojan）**：名字来源于“特洛伊木马”。它伪装成正常的软件（比如一个游戏或播放器），一旦你安装运行，它就会在后台偷偷打开一扇门，让黑客潜入你的电脑偷取密码。
*   **勒索软件（Ransomware）**：极度恶劣！它会悄悄把你的所有照片和文档锁死（加密），然后弹出一个倒计时窗口，要求你必须支付比特币（无法追踪的虚拟货币）才给你解开，否则销毁文件。**防范它的唯一好办法就是：定期备份数据。**
*   **钓鱼网站（Phishing）**：做一个长得和银行、淘宝一模一样的假网站。你以为在登录，其实是把账号密码直接发送给了骗子。

### 6.2 密码安全：如何设置一个"猜不到又记得住"的密码？

**最糟糕的密码**：`123456`、`password`、`你的名字拼音+生日`。黑客用字典软件，几秒钟就能破解。

**好密码的原则**：
1.  **足够长**：长度比复杂性更重要。至少 8 位，12 位以上更好。
2.  **不共用**：千万不要所有网站用同一个密码！万一某个小网站被黑客攻破，你的微信、支付宝也会跟着遭殃。

**设置密码的小妙招（句子记忆法）**：
想一句话：`我每天早上7点半起床跑步` -> 拼音首字母：`wmtz7dbqcpb`。
然后根据不同网站加点变化，比如微信：`Wx_wmtz7dbqcpb`，淘宝：`Tb_wmtz7dbqcpb`。
这样既长、又复杂、还不容易忘，且每个网站都不一样。

### 6.3 双重验证（2FA）：给账号加一把“防盗锁”

这是目前**最有效**的防盗号方法。

开启双重验证后，在陌生设备上登录账号时，不仅需要输入**第一重（密码）**，还必须输入**第二重（你手头的临时证明）**，比如：
*   发到你手机上的短信验证码
*   微信里弹出的确认提示
*   指纹或刷脸

这样一来，即使黑客在地球另一端偷到了你的密码，只要他没有拿到你的手机，依然登不进去。**重要的账号（微信、支付宝、邮箱）强烈建议去设置里开启它！**

### 6.4 识别诈骗：那些套路你中过几个？

网络诈骗千变万化，但核心往往利用人性的弱点：**贪婪** 和 **恐惧**。

*   **“你中奖了/你的快递丢了要理赔”**（贪婪）：凡是让你先交点手续费，或者点不明链接填银行卡号的，100%是诈骗。
*   **“我是公安局/查到你涉嫌洗钱”**（恐惧）：警察绝不会通过电话办案，也不会让你把钱转到所谓的“安全账户”。
*   **“老板/熟人微信借钱”**：骗子可能盗取了你朋友的微信号。凡是网上借钱，**必须打个语音或视频电话确认是本人**。

**防骗口诀：天上不会掉馅饼，要钱一律不答应；不明链接不要点，转账之前要核实。**

### 6.5 公共Wi-Fi的风险：咖啡店里的"偷听者"

在机场、咖啡店连上了免费的、不需要密码的 Wi-Fi？请小心！

黑客可以轻易地截获同一个 Wi-Fi 网络里其他人传输的数据。如果在这种不安全的网络下，你登录了网银或输入了密码，黑客就能“偷看”到。

**建议**：在公共免费 Wi-Fi 下，可以看看视频、刷刷新闻。但如果涉及到**转账、付款或登录重要账号**，请立刻断开 Wi-Fi，使用自己手机的 4G/5G 流量，那要安全得多。

### 6.6 隐私设置：手机和电脑的权限管理

你的手机可能在“监视”你。很多 APP 都会过度索取权限。

*   请定期检查手机系统的“权限管理”。
*   **麦克风/摄像头**：只给需要语音通话或拍照的 APP（如微信、相机）授权，其他的关掉。
*   **相册**：购物软件不让你发评价就不让用？可以设置成“仅允许访问选中的照片”，而不是全部照片。
*   **剪贴板**：如果你刚复制了一段文字，打开某 APP 它就弹出来相关内容，说明它在读取你的剪贴板，可以在系统里关闭它的读取权限。

### 6.7 遇到安全问题怎么办？应急处置手册

*   **手机丢了**：立刻借别人手机拨打运营商电话（如10086）**挂失手机卡**（最重要，防止骗子接收验证码）！然后冻结微信和支付宝账号。
*   **电脑中勒索病毒了**：立即**拔掉网线/关掉Wi-Fi**（防止它感染局域网里其他电脑），不要向黑客付钱（付了也不一定给你解开），找专业人士重装系统。
*   **不小心点了钓鱼链接并填了密码**：立刻立刻去**修改密码**。如果填了银行卡，立刻打银行客服冻结卡片。
