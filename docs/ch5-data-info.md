# 第五章：数据与信息——数字世界的"原材料"

在数字世界里，所有的文字、照片、语音、电影，本质上都是“数据”。今天，我们来认识一下这个数字世界的原材料，以及如何更好地保护它。

### 5.1 什么是数据？什么是信息？

*   **数据（Data）**：是客观事物的记录，是冰冷的数字。比如：`39`。
*   **信息（Information）**：是经过处理、对人有意义的数据。比如：`体温 39℃，发烧了`。

计算机非常擅长处理海量的数据，而人类则擅长从这些数据中提取有价值的信息。

### 5.2 比特（bit）与字节（Byte）：数字世界的最小单位

计算机底层是个电子设备，它其实是个“文盲”，它不认识中文也不认识英文，它只认识两种状态：**通电（1）** 和 **断电（0）**。

*   **比特（bit，用小写 `b` 表示）**：也就是一个 `0` 或一个 `1`。这是数字世界**绝对的最小单位**。
*   **字节（Byte，用大写 `B` 表示）**：1个字节等于8个比特（`1 Byte = 8 bits`）。**字节是用来计算文件大小的基础单位**。
    *   在英文中，1个字母（如 'A'）通常占 1个字节。
    *   在中文中，1个汉字通常占 2到3个字节。

### 5.3 KB / MB / GB / TB：容量单位的换算

为了表示大文件，我们在字节（Byte）前面加上了表示数量级的字母（就像“克”和“千克”）：

*   **1 KB** (Kilobyte) ≈ 1,000 字节（大约是一篇几百字的短文）
*   **1 MB** (Megabyte) ≈ 1,000 KB（大约是一张高清照片，或一首普通音质的歌）
*   **1 GB** (Gigabyte) ≈ 1,000 MB（大约是一部两小时的超清电影）
*   **1 TB** (Terabyte) ≈ 1,000 GB（现在买个新电脑，硬盘往往是 1TB）

> *注：严格的计算机科学中是按照 1024 来换算的（1KB = 1024B），但为了方便记忆，生活和商业宣传中通常近似按 1000 计算。这就解释了为什么你买了一个“512GB”的手机，插上电脑显示只有 470GB 左右，这不是被骗了，而是换算方式不同。*

### 5.4 数据是怎么存储的？本地 vs 云端

*   **本地存储**：数据保存在你**手头设备**的硬盘里（比如你的手机、电脑、U盘）。
    *   **优点**：不需要网络就能看，绝对掌握在自己手里。
    *   **缺点**：一旦设备丢失、进水或损坏，数据可能永远找不回来。
*   **云端存储（云盘）**：数据通过网络，传到了大公司（如苹果、百度、阿里云）位于机房的**巨型服务器硬盘**里。
    *   **优点**：只要有网，用任何设备登录账号都能看；不怕自己手机丢了。
    *   **缺点**：没有网络就用不了；如果账号密码泄露，数据会被别人看到。

### 5.5 备份的重要性：鸡蛋不要放在一个篮子里

数字世界有一条铁律：**任何没有备份的数据，都早晚会丢失**。

硬盘是消耗品，总有一天会坏；手机也有不小心摔坏或丢失的可能。对于那些**“一旦丢失就会让你痛哭流涕”**的数据（如孩子的成长照片、多年的工作文档），一定要做备份。

**推荐的“3-2-1 备份法”**（简化版）：
*   **存一份在电脑或手机里（本地）**
*   **再存一份在移动硬盘或U盘里（本地物理隔离）**
*   **再传一份到可靠的网盘里（云端异地）**

### 5.6 你的数据是有价值的——个人信息保护意识

在大数据时代，你的浏览记录、购物偏好、所在位置，都是互联网公司眼中的“黄金”。他们通过收集这些数据，给你精准推送广告，甚至“杀熟”。

**保护个人信息的小习惯：**
1.  **少填真实信息**：在不重要的网站注册时，没必要填真实的姓名和生日。
2.  **关闭不必要的权限**：新装一个美颜相机APP，它如果非要读取你的“通讯录”和“地理位置”，直接拒绝。
3.  **谨慎连免费Wi-Fi**：有些不要密码的免费Wi-Fi，专门用来监听你传输的数据。
4.  **快递单不要直接扔**：扔快递盒前，把名字和电话涂黑或撕碎，防止被推销电话盯上。
