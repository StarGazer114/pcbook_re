### 附录A：常用术语中英文对照表（Glossary）

| 英文缩写 / 单词 | 中文含义 | 通俗解释 |
| :--- | :--- | :--- |
| **PC** (Personal Computer) | 个人计算机 | 就是我们常说的电脑（包括台式机和笔记本）。 |
| **OS** (Operating System) | 操作系统 | 电脑的“大管家”，如 Windows, macOS。 |
| **APP** (Application) | 应用程序 | 手机或电脑上的软件，如微信、Word。 |
| **CPU** (Central Processing Unit) | 中央处理器 | 电脑的“大脑”，负责计算。 |
| **RAM** (Random Access Memory) | 运行内存 | 电脑干活时的“桌面”，越大越不卡。 |
| **ROM / Storage** | 存储器 / 硬盘 | 电脑存东西的“仓库”，断电不丢失。 |
| **UI** (User Interface) | 用户界面 | 你在屏幕上看到的按钮、图标、颜色等设计。 |
| **URL** | 统一资源定位符 | 俗称“网址”或“链接”，如 `www.baidu.com`。 |
| **Bug** | 程序漏洞 | 原意是“虫子”，指软件里的错误或缺陷导致闪退、卡死。 |
| **AI** (Artificial Intelligence) | 人工智能 | 让机器模仿人类思考的技术。 |
| **IoT** (Internet of Things) | 物联网 | 让电器、汽车等万物连上网。 |

### 附录C：常见问题 Q&A

**Q：电脑用久了很卡，需要每天用“安全卫士”清理垃圾吗？**
> A：**不需要。** 现在的 Windows 10/11 系统已经足够聪明，内置了安全中心和清理功能。所谓的第三方“安全管家”往往自身就是让电脑变卡的罪魁祸首（后台占用高、弹窗多）。只要你不乱点网页瞎下软件，电脑“裸奔”（只用系统自带安全中心）是完全没问题的。

**Q：手机充电一整晚不拔，会损坏电池吗？**
> A：**不会。** 现在的智能手机电池和充电器都有电源管理芯片，充满 100% 后会自动切断电流。真正伤害锂电池的是：经常把电用到 0% 自动关机，以及在极度高温/低温下玩手机。建议电量低于 20% 就赶紧充。

**Q：我的微信密码忘了，手机号也换了，怎么找回？**
> A：打开微信登录界面，点击“找回密码” -> “申诉找回微信账号密码”。按系统提示操作，通常需要邀请两个微信好友辅助验证，这是目前唯一官方的找回方式。

### 附录D：快捷键速查手册

掌握快捷键，能让你的电脑操作效率翻倍！

**Windows 系统最常用的 8 个快捷键：**
*   `Ctrl` + `C` ：复制 (Copy)
*   `Ctrl` + `V` ：粘贴 (Paste)
*   `Ctrl` + `X` ：剪切 (Cut，把东西移走)
*   `Ctrl` + `Z` ：撤销 (Undo，刚写错字了，按一下就能恢复)
*   `Ctrl` + `S` ：保存 (Save，写文档时养成随手按它的好习惯)
*   `Ctrl` + `A` ：全选 (Select All)
*   `Ctrl` + `F` ：查找 (Find，在长文章里找特定的词)
*   `Alt` + `Tab` ：在打开的多个软件窗口之间快速切换

*(如果你用的是苹果 Mac 电脑，只需把上面所有的 `Ctrl` 键换成键盘上的 `Command` 键 `⌘` 即可。)*
