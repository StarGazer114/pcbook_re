# 第七章：人工智能（AI）——会"思考"的机器

近年来，“人工智能”（AI）这个词铺天盖地。有人说它会让人类失业，有人说它能改变世界。它到底是个什么神奇的东西？

### 7.1 什么是人工智能？和科幻电影里的一样吗？

*   **人工智能（Artificial Intelligence, AI）**：简单来说，就是让计算机程序去**模仿人类的智能行为**（如学习、推理、感知、理解语言）。
*   **和科幻电影的区别**：电影里的AI（如《终结者》或《流浪地球》里的MOSS）属于**“强人工智能”**，它们有自我意识和情感，目前这种技术还远远不存在。我们现在用的都是**“弱人工智能”**，它们只是在**某一个特定领域**（如下棋、画画、聊天）非常厉害，但它并不真的知道自己在干什么。

### 7.2 机器学习：AI是怎么"学会"的？

以前，人类怎么让电脑干活？**写死规则**。比如：如果输入1+1，就输出2。
现在，人类怎么教AI干活？**机器学习（Machine Learning）**。

*   **生活类比**：就像教小孩子认猫。你不需要告诉他“猫有两只耳朵四条腿”（因为狗也有）。你只需要给他看一万张猫的照片和一万张狗的照片，告诉他哪些是猫，哪些是狗。看多了，他（AI模型）自己就能总结出猫的特征。当你再拿一张新照片给他看时，他就能大概率猜出是不是猫。

这就是为什么AI需要大量的**“数据”**来训练。数据就是喂给AI的“饲料”。

### 7.3 你每天都在用的AI：它早就在你身边了

很多人以为ChatGPT才是AI，其实你早就离不开AI了：
*   **抖音/淘宝的推荐算法**：AI在悄悄学习你的喜好，推测出你爱看什么。
*   **人脸识别/指纹解锁**：AI在对比你现在的脸和当初存进去的脸是不是同一个人。
*   **手机语音助手（Siri / 小爱同学）**：AI在把你的声音转成文字，并尝试理解你的命令。
*   **照片自动分类**：手机相册能自动把“风景”、“人物”分类，也是AI的功劳。

### 7.4 大语言模型（ChatGPT等）：它是怎么回答问题的？

像 ChatGPT（或国内的文心一言、通义千问）这样的“大语言模型”，看起来像个无所不知的神童。

*   **它的原理其实是“文字接龙”**。
*   开发者把全人类互联网上几乎所有的文字资料（几千亿个网页、书籍）都喂给了它。它通过庞大的计算，找出了人类语言的概率规律。
*   当你说“床前明月”，它通过概率计算得出，下一个字极大概率是“光”。当你的问题很复杂时，它依然在进行极度复杂的“文字接龙”，拼凑出最符合逻辑的回答。
*   **注意：它并不理解自己说的话的意思，它只是在预测哪个词最该出现在下一个位置。**

### 7.5 AI能做什么，又有哪些局限？

*   **擅长**：
    *   快速总结长篇文章。
    *   写公文、邮件、润色文章（当你的打字员）。
    *   画插图、做海报（文生图）。
    *   写代码、找代码里的错误。
*   **局限（目前）**：
    *   **会“一本正经地胡说八道”（机器幻觉）**：因为它只是在做概率接龙，有时候它会为了顺口，编造出根本不存在的事实。**所以，绝不能完全信任AI的答案，尤其是在医疗、法律等严谨领域。**
    *   缺乏真正的情感和道德判断。

### 7.6 AI时代，我们该怎么和它相处？

AI就像是以前发明的“拖拉机”或“计算器”。它是一个极其强大的**工具**。

*   **不要害怕被AI取代**，但要警惕被**“会用AI的人”**取代。
*   **学会提问（Prompt）**：未来，能精准地向AI提出好问题、给出好指令，将成为一项核心技能。
*   把它当成一个不知疲倦的**实习生**。让它去干粗活（查资料、写初稿），你来做最后的老板（审核、修改、做决策）。
