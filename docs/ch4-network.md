# 第四章：网络世界——连接一切的"蜘蛛网"

单台计算机虽然强大，但如果没有网络，它就像一座“信息孤岛”。正是因为有了网络，我们才能看视频、点外卖、和远方的朋友视频通话。

### 4.1 什么是互联网（Internet）？

互联网，本质上就是**把全世界所有的计算机，用粗细不一的网线（光缆、海底电缆）和无线电波连接起来的一张巨大的“蜘蛛网”**。

当你用手机看一篇网上的文章时，其实是远在几千公里外的一台超级计算机（服务器），顺着这张网，把文章的文字和图片“发送”到了你的手机屏幕上。

### 4.2 网络是怎么传输数据的？（快递类比）

网络传输数据的方式，和现实生活中的**寄快递**非常像。

假设你要给朋友发一张 5MB 的照片：
1.  **打包与拆分**：网络不会一次性把整张照片扔过去。它会把照片拆成成千上万个极小的“数据包”（就像把大件物品拆成好几个小快递盒子）。
2.  **贴上地址**：每个数据包都会贴上“收件人”和“寄件人”的地址（IP地址）。
3.  **路由传输**：这些数据包顺着网线出发，路上会经过很多个交叉路口（路由器）。路由器就像是快递中转站，负责指引数据包走向正确的下一站。
4.  **组装**：等所有数据包都到达朋友的手机后，手机会把它们重新拼装成那张完整的照片。

### 4.3 Wi-Fi、宽带、流量：连上网的三种方式

*   **宽带（有线网络）**：运营商（电信/联通/移动）拉到你家里的一根实实在在的网线。速度最快、最稳定。
*   **Wi-Fi（无线局域网）**：家里的“路由器”把宽带的网线信号，转换成了无线电波。它的范围很小，一般只能覆盖一个屋子。
*   **移动数据（流量，如4G/5G）**：手机直接连接到室外的信号基站。不需要路由器，走哪连哪，但按使用量（流量）收费。

### 4.4 IP地址与域名：门牌号与门店名

要在网络世界里准确地找到一台计算机，需要地址。

*   **IP地址**：网络世界里的**“门牌号”**。它是一串由数字组成的号码，比如 `192.168.1.1`。每台连上网的设备都有一个唯一的IP地址。
*   **域名（Domain Name）**：因为一长串数字太难记了，人们发明了域名（**“门店名”**）。比如 `www.baidu.com`。
*   **DNS（域名系统）**：就像是网络里的**“114查号台”**。当你在浏览器输入 `baidu.com` 时，DNS会立刻查出它对应的IP地址，然后带你过去。

### 4.5 浏览器与搜索引擎：怎么在网上"找东西"

*   **浏览器**（如 Chrome、Edge、Safari、微信内置浏览器）：是你用来**“逛网页”的工具**。就像是一辆带你畅游互联网的汽车。
*   **搜索引擎**（如 百度、谷歌、必应）：是网页里的一个**具体服务**。当你不知道某个网站的域名时，你可以告诉搜索引擎你要找什么，它会给你列出一堆相关的网页链接。就像是商场里的“导购台”。

### 4.6 HTTP vs HTTPS：安全的门锁

仔细看浏览器上方的网址，一般是以 `http://` 或 `https://` 开头。

*   **HTTP**：就像寄**明信片**。所有传输的信息（包括你输入的密码）都是明文的，路上的邮递员（黑客）想看就能看。
*   **HTTPS**（多了一个S，Secure）：就像寄**带锁的保险箱**。你的信息会被加密，只有到了目的地才能解开。**现在绝大多数正规网站都使用 HTTPS。在输入密码或付款时，一定要确保网址开头是 HTTPS（或者浏览器地址栏有一把小锁头标志）。**

### 4.7 带宽、网速、延迟：三个影响体验的指标

为什么有时网速很快，玩游戏却还是卡？
*   **带宽（Bandwidth）**：就像是**马路的宽度**。比如“百兆宽带”，代表马路很宽，能同时跑很多车。
*   **网速（下载/上传速度）**：就像是**一秒钟能通过多少辆车**。看超清视频需要高网速。
*   **延迟（Latency / Ping）**：就像是**一辆车从你家开到目的地再回来的时间**。玩对战游戏时，延迟比网速更重要。如果延迟高，你开了一枪，可能半秒后服务器才收到，这时候你早就被击中了。
